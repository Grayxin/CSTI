# 角色设定
你是一位精通原生 Web 开发的顶级前端架构师兼资深 UI/UX 设计师。

# 任务目标
请基于项目中的 3 个文件（《题目.md》、《性格.md》、《评分标准.md》），开发一个“CSTI 公务员性格测试”网页应用。

# 开发要求
1. **文件分离**：请将代码拆分为三个独立的文件：`index.html`（结构）、`style.css`（样式）和 `script.js`（逻辑）。
2. **数据驱动**：请将《题目.md》中的 30 道题目和《性格.md》中的 25 种性格解析完整地转为 `script.js` 中的常量数据。
3. **算法实现**：必须严格执行《评分标准.md》中的计分逻辑：
   - 统计 L/D、O/S、N/F、P/A 八个维度的得分。
   - 实现“胜者通吃”对抗逻辑及“同分降级原则”。
   - **核心**：优先判定“极端阈值熔断”（单项得分 >= 12）和“隐藏彩蛋（薛定谔的临时工）”。

# 【ui-ux-pro-max 技能】深度优化指令（极其重要）
为了达到顶级体验，请在视觉和交互上严格执行以下标准：
1. **视觉呈现 (CSS)**：
   - 采用移动端优先的响应式设计（Mobile First），并在大屏设备上居中显示为精美的卡片式容器。
   - 严格参考上传的截图：首页深色渐变 + 抽象山峰背景；答题页/结果页使用米白色/浅橙色调，主打高级留白和干净的阅读体验。
   - 引入高质量字体（如系统默认的无衬线字体栈），精确控制字重（Font Weight）、行高（Line Height）和字间距，提升文案的可读性。
   - 卡片和按钮必须拥有细腻的拟物阴影（Soft Box Shadow）和圆角（Border Radius）。
2. **交互动效 (JS/CSS)**：
   - 全局禁用生硬的突变！所有状态切换、按钮悬浮（Hover/Active）必须附带平滑的贝塞尔曲线过渡动画（Transition: all 0.3s cubic-bezier(...)）。
   - 答题体验极度丝滑：点击选项后，选项出现高亮选中状态，延迟 300ms 后页面伴随柔和的淡出淡入动画（Fade in/out）自动切换至下一题。
   - 采用单页面应用（SPA）逻辑，`index.html` 内三个视图的切换必须无缝且带有微动效。
3. **数据可视化增强**：
   - 结果页必须引入 CDN 版的 `Chart.js`，绘制一个高精度的五维/四维雷达图。
   - 雷达图的网格线、填充色必须与主题色（浅橙色）保持完美一致，并带有半透明渐变效果。
4. **占位符美化**：结果页的性格图片，请使用富有设计感的 CSS 图形或 Lucide 图标进行精美占位。
5. **ui-ux-pro-max优化**：所有页面必须使用ui-ux-pro-max 技能进行页面优化
